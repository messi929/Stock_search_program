Stock Screener Pro(이하 "서비스")는 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」·「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 및 GDPR 등 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 수집하고, 어떻게 이용·보관·공유하는지 설명합니다.

1. 수집하는 정보 (Information We Collect)

1.1 회원가입·로그인 정보

  • 이메일 주소
  • 사용자 식별자 (Firebase UID)
  • 프로필 이름·프로필 이미지 URL (Google 로그인 시 제공되는 공개 정보)

1.2 결제 정보

결제는 Lemon Squeezy Inc.가 직접 처리하며, 운영자는 이용자의 카드번호 등 결제 상세정보를 저장·조회하지 않습니다. 운영자는 결제 결과로 다음 정보만 수신·저장합니다:

  • Lemon Squeezy Customer ID
  • 구독 ID, 구독 상태, 다음 결제일, 플랜명
  • 결제일시 및 금액 (통계 목적)

1.3 서비스 이용 정보

  • 접속 IP, 브라우저·OS 정보(User-Agent), 접속 일시
  • 활성 세션 정보 (동시 접속 제한을 위한 세션 ID 및 최근 활동 시각)
  • 관심종목, 필터 설정, 알림 설정
  • 서버 로그 (오류 추적, 보안 목적)

1.4 어뷰징 방지를 위한 로그인 기록

계정 공유·무한 체험 등 서비스 약관 위반 탐지 목적으로, 로그인 시 다음 정보를 기록합니다:

  • 로그인 IP 및 접속 국가, User-Agent, 타임스탬프
  • 30일 기준 unique IP 개수 집계 (기준치 초과 시 관리자 검토 대상 플래그)
  • 동시 접속 세션 수 (기본 허용 2개 · 초과 시 가장 오래된 세션 자동 종료)

정상적인 사용(집/회사/모바일 등 여러 위치 접속)은 문제되지 않으며, 반복적 이상 패턴이 감지된 계정에 한해 관리자가 개별 검토합니다. 관련 기록은 최대 6개월 보관 후 자동 삭제됩니다.

  • 서비스 제공 및 계정 관리
  • 구독·결제 상태 확인 및 과금

  • 고객 문의 응대, 이용약관 위반 대응
  • 중요 공지 및 약관 변경 안내 (이메일)
  • 법령상 의무 이행

3. 정보 보관 기간 (Retention)

  • 회원 정보: 회원 탈퇴 시 즉시 삭제 (단, 법령에 따라 보관이 요구되는 정보 제외)

운영자는 이용자의 동의 없이 개인정보를 제3자에게 판매·대여하지 않습니다. 다음과 같은 목적으로 제한적으로 공유·위탁됩니다:

제공자 목적 제공 정보
Google LLC (Firebase Auth) 로그인 인증 이메일, UID
Google LLC (Firestore) 계정·구독·설정 데이터 저장 UID, 구독 정보, 설정값
Google Cloud Run (asia-northeast3) 서비스 호스팅 접속 로그, 요청 데이터

각 제공자의 개인정보처리방침:
