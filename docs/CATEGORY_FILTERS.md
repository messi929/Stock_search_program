## 정량 전략 (Strategy) — 7개

### 급등 예보 (`surge`) — Phase 3
> 곧 급등할 가능성이 높은 종목을 포착

| 조건 | 값 | 설명 |
|------|-----|------|
| 시가총액 | ≥ 500억 | 소형주 제외 |
| 거래대금 | ≥ 10억 | 유동성 확보 |
| **pre_surge_only** | `is_pre_surge == 1` | **급등예보 점수 3/5 이상** |

**급등예보 점수 산출 (0~5점, ≥3점 시 시그널):**
1. 거래량비 1.5~5.0x + 가격변동 < 3% (매집 징후) → +1
2. 거래량 2일 이상 연속 증가 → +1
3. 이평선 수렴 (MA5-MA60 스프레드 < 3%) → +1
4. 연속 상승 1~3일 (초기 상승) → +1
5. 골든크로스 발생 → +1

**정렬**: pre_surge_score 내림차순
**사용자 필터**: 시총, 거래량비, 등락률

| 조건 | 값 | 설명 |
|------|-----|------|
| **buy_score** | ≥ 30점 | 중간 구간 이상 |
| 시가총액 | ≥ 1,000억 | |
| 거래대금 | ≥ 10억 | |

**종합 점수 산출 (0~100점):**

| 카테고리 | 배점 | 세부 |
|---------|------|------|
| **기술적 시그널** | 40점 | 급등예보(15) + 돌파점수(10) + 골든크로스(8) + 정배열(7) |
| **모멘텀** | 25점 | RSI 반등구간(10) + 거래량추세(8) + 연속상승(7) |
| **수급** | 20점 | 외국인 순매수(8) + 기관 순매수(7) + 매집시그널(5) |
| **가치** | 15점 | PER 적정성(8) + PBR 저평가(7) |

**구간 등급 기준 (buy_grade — 중립 라벨):**
- 상위: ≥ 70점
- 준상위: ≥ 50점
- 중간: ≥ 30점
- 관찰: < 30점 (노출 안 됨)

**정렬**: buy_score 내림차순
**사용자 필터**: 시총, PER, 거래량비, 시장

## 매매 시그널 (Signal) — 4개

### 매집 의심 (`accumulation`) — Phase 3
> 조용히 물량을 모으고 있는 것으로 의심되는 종목

| 조건 | 값 | 설명 |
|------|-----|------|
| **accumulation** | `== 1` | 매집 시그널 활성 |
| 시가총액 | ≥ 500억 | |
| 거래대금 | ≥ 10억 | |

**매집 시그널 조건:**
- 거래량비 ≥ 1.5x (20일 평균 대비)
- 가격변동 < 2% (조용한 가격)
- 두 조건 동시 충족 시 `accumulation = 1`

**정렬**: 거래량비 내림차순
**사용자 필터**: 시총, 거래량비, 등락률

### 돌파 임박 (`breakout`) — Phase 3
> 신고가를 돌파할 가능성이 높은 종목

| 조건 | 값 | 설명 |
|------|-----|------|
| 52주 고가 대비 | ≥ -5% | 고가 근접 |
| **돌파점수** | ≥ 2/4 | 최소 2개 조건 충족 |
| 시가총액 | ≥ 500억 | |
| 거래대금 | ≥ 10억 | |

**돌파점수 산출 (0~4점):**
1. 52주 고가 5% 이내 → +1
2. 거래량 2일 이상 연속 증가 → +1
3. MA 정배열 (MA5 > MA20 > MA60) → +1
4. 골든크로스 (MA5 > MA20) → +1

**정렬**: breakout_score 내림차순
**사용자 필터**: 시총, 거래량비, 등락률
