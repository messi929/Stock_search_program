# Stock Screener Pro — 기술 문서 (v4.0.0)

## 1. 프로젝트 개요

KOSPI/KOSDAQ/NASDAQ/S&P500 + ETF 전종목을 대상으로 15개 카테고리별 **정량 스크리닝**을 제공하는 웹 종목 분석 도구.
"이미 오른 종목"이 아닌 **"지금 주목할 구간의 종목"**을 실시간 시그널 기반으로 탐색. 매수 권유가 아닌 참고 정보를 제공.

### 핵심 철학
- **예측 > 결과**: 급등 후가 아니라 급등 전 징후를 포착
- **매집 감지**: 거래량↑ + 가격↓ = 세력 매집 패턴 관찰
- **타이밍**: 골든크로스 진입, 과매도 구간, 돌파 임박 시점 포착

**기술 스택**: Python 3.11 / FastAPI / FinanceDataReader / 네이버금융 크롤링 / Vanilla JS / lightweight-charts / Pretendard Font

## 4. 데이터 흐름

### 3단계 로딩

| Phase | 내용 | 소요 시간 | 활성화 |
|-------|------|----------|--------|
| **Phase 1** | KR 스냅샷 + ETF + US 스냅샷 | ~5초 (캐시 시 1초) | 전체 15 카테고리 즉시 |
| **Phase 2** | PER/PBR (상위 500, 8스레드) + 테마 + 배당지속성 + 외국인/기관 | ~2분 (캐시 시 1초) | 결과 정밀화 |
| **Phase 3** | 히스토리 OHLCV (상위 400, 5스레드) → MA/RSI/52주 | ~1분 (캐시 시 1초) | 예측 시그널 정밀화 |

### 데이터 소스

| 소스 | 제공 데이터 | 방식 |
|------|-----------|------|
| **FinanceDataReader** | KR/US 주식 OHLCV, 시총, ETF | `StockListing()`, `DataReader()` |
| **네이버금융** | PER/PBR/배당률, 배당 히스토리 | HTML 크롤링 (8스레드 병렬) |
| **네이버금융** | 테마 분류 + 종목-테마 매핑 | HTML 크롤링 (최대 50페이지) |
| **네이버금융** | 외국인/기관 순매수 | HTML 크롤링 (KOSPI+KOSDAQ) |
| **자체 계산** | MA, RSI, 52주, 급등예보, 매집신호, 돌파점수 | pandas 기반 |

### 캐싱
- Parquet: snapshot, etf, us_snapshot, history
- JSON: fundamentals, themes, stock_themes, foreign_inst, dividend_history
- 유효기간: 당일만 (날짜 변경 시 자동 만료)

### 자동 갱신
- 주기: 30분 (`REFRESH_INTERVAL_MIN`)
- 조건: 평일 09:00~15:30 (장 운영시간)
- 방식: asyncio 백그라운드 태스크
- 알림: WebSocket push → 프론트엔드 자동 갱신

## 5. 카테고리 체계

### 5.1 정량 전략 (Strategy) — 6개

| 카테고리 | 키 | 목적 | 핵심 조건 | 정렬 |
|---------|----|----|---------|------|
| **급등 예보** | `surge` | 급등 직전 매집 종목 | 시총≥500억 | 급등예보점수↓ |
| **성장주** | `growth` | 고ROE 합리적 밸류 | ROE≥15%, PER 0.1~30, 시총≥1000억 | ROE↓ |
| **저평가주** | `value` | PER/PBR 바닥권 | PER 0.1~10, PBR 0.1~1.0, 시총≥1000억 | PER↑ |
| **배당주** | `dividend` | 고배당 안정 수익 | 배당률≥3%, 시총≥1000억 | 배당률↓ |
| **추세 진입** | `momentum` | 골든크로스 진입 시점 | 골든크로스, 시총≥500억 | 돌파점수↓ |
| **반등주** | `turnaround` | 과매도 바닥 반등 | RSI≤35, 거래량비≥1.5x, PBR≤1.5 | RSI↑ |

### 급등 예보 점수 (pre_surge_score, 0~5)
| 조건 | 점수 | 의미 |
|------|-----|------|
| 거래량비 1.5~5x + 가격변동 < 3% | +1 | 매집 징후 |
| 거래량 연속 증가 2일+ | +1 | 수급 유입 |
| 이평선 수렴 < 3% | +1 | 에너지 축적 |
| 연속 상승 1~3일 | +1 | 초기 상승 |
| 골든크로스 또는 정배열 | +1 | 기술적 전환 |

### 5.2 시장별 (Market) — 5개

| 카테고리 | 키 | 조건 | 정렬 |
|---------|----|----|------|
| **ETF** | `etf` | ETF만 | 거래량↓ |
| **대형주** | `bluechip` | 시총≥50,000억 | 시총↓ |
| **중소형주** | `smallcap` | 시총 500~5,000억 | 거래량비↓ |
| **테마주** | `theme` | 시총≥300억 + 테마 칩 셀렉터 | 거래량비↓ |
| **관심종목** | `watchlist` | 사용자 localStorage 저장 | 등락률↓ |

### 5.3 패턴 시그널 (Signal) — 4개

| 카테고리 | 키 | 목적 | 핵심 조건 | 정렬 |
|---------|----|----|---------|------|
| **매집 의심** | `accumulation` | 거래량↑ 가격↓ 세력 매집 | 시총≥500억 | 거래량비↓ |
| **스마트머니** | `foreign_inst` | 외국인·기관 순매수 | 시총≥1000억 | 외국인순매수↓ |
| **돌파 임박** | `breakout` | 52주 고가 5% 이내 | vs_high_52w≥-5%, 시총≥500억 | 돌파점수↓ |
| **과매도 구간** | `oversold` | RSI≤35 과매도 관찰 | 시총≥500억 | RSI↑ |

| 파라미터 | 설명 |
|---------|------|

| `rsi_min/max` | RSI 필터 |
| `theme` | 테마명 필터 |
| `tickers` | 특정 종목 (쉼표 구분, 관심종목용) |

├─ 테마 칩 바 (테마주 선택 시만 표시) ───────────────────────┤
├─ 필터 카드 (접이식) ───────────────────────────────────────┤
├─ 결과 헤더 (종목 수 + CSV/Excel) ─────────────────────────┤

### 디자인 시스템
- **테마**: 다크 (#0b0e14 배경)
- **폰트**: Pretendard (CDN)
- **컴포넌트**: 칩(pill), 세그먼트 컨트롤, 카드 + 그림자
- **색상**: 상승 빨강 / 하락 파랑 (한국 관례)
- **애니메이션**: 칩 press scale(0.97), 모달 slideUp, 행 fadeIn

### 주요 기능
- 국내/해외 원클릭 전환
- 카테고리 칩 선택 → 즉시 결과
- 테마주 서브 셀렉터 (AI, 2차전지, 바이오 등)
- ★ 관심종목 (localStorage)
- 종목 클릭 → 상세 모달 + 캔들차트
- CSV/Excel 내보내기
- 알림 설정 (등락률, 급등주)
- WebSocket 실시간 갱신

## 9. 예측 시그널 지표

| 지표 | 설명 | 계산 |
|------|------|------|
| `pre_surge_score` (0~5) | 급등 예보 점수 | 매집+거래량추세+이평수렴+연속상승+골든크로스 |
| `accumulation` (0/1) | 매집 신호 | 거래량비≥1.5x AND 가격변동<2% |
| `breakout_score` (0~4) | 돌파 임박 점수 | 52주고가근접+거래량추세+정배열+골든크로스 |
| `volume_trend` | 거래량 연속 증가 일수 | 최근 5일 거래량 변화 |
| `ma_squeeze` (%) | 이평선 수렴도 | \|MA5-MA60\|/MA60×100, 낮을수록 에너지 축적 |
| `is_pre_surge` (0/1) | 급등 예보 여부 | pre_surge_score ≥ 3 |
