## 1. 로그인 개선 3종

- **구글 속도**: 로그인 완료 판정이 프로필 조회까지 블로킹 + 컴포넌트마다 독립 fetch → `useUserProfile`을 TanStack Query 전역 캐시로 통합(호출처 6곳 무수정).
- **카카오**: Firebase native 미지원 → `POST /api/auth/kakao`(인가코드→카카오 token/user→`create_custom_token`) + `/login/kakao/callback`. 앱 ID 1476310, REST 키 주입·재배포·인가 검증 완료.
- **이메일**: login 3모드(로그인/회원가입/비번재설정) + 인증메일·재발송. 트라이얼은 `email_verified` 필수 + 일회용도메인 차단(+28)·IP 누적 제한(24h 1개/30일 2개)으로 어뷰징 방어. Firebase Email provider 활성 확인.

## 2. UX 강화

- **스크리너 KR/US 탭**: 백엔드 `?market=` 이미 지원 → 결과 페이지에 🇰🇷/🇺🇸 탭(US 743종목 실재 확인).
- **분석 백그라운드 지속**: `AnalyzeView`가 언마운트 시 SSE를 끊던 것 → `analysisStore`(zustand)로 ticker별 상태+AbortController 전역화. 화면 전환해도 계속, 재진입 복원.
- **진행중 표시기**: 대시보드 상단 배너(running 분석). **최근 분석**: recents(localStorage persist) 카드.
- **AI 사용량 이력**: analyze/validate/discover 시 `users/{uid}/analysis_history` 기록 → 사용량 항목 클릭 시 `/history` 전용 페이지(유형 탭+일자별). 종목명(snapshot 조인)·**그 시점 결과 요약+당시가**(분석 완료 시 `_update_history_result`)까지 표시.
- **랜딩**: Hero 2단(카피/CTA + 제품 미리보기 목업) + 신뢰배지. 로그인 시 홈/로고가 `/dashboard` 직행(`LandingRedirect`).
- **검색 드롭다운 잘림**: 기본 Card가 `overflow-hidden`이라 absolute 드롭다운이 잘리던 것 → SearchTab Card에 `overflow-visible`.
