## 1. 유닛 이코노믹스 — 적자구조 해결 (docs/axis/UNIT_ECONOMICS.md)

**조치 (전부 배포)**:
- **Quota enforce**(`api/routes/ai.py`): `_enforce_quota`/`_count_month_usage` 신설, `/analyze`·`/discover` 차단. Free20/Pro100/Premium300. + `get_usage` 평면키(`agents.x.calls`) 파싱 버그 수정(항상 0 표시되던 문제).
- **Strategist Opus→Sonnet**: A/B 검증(`scripts/strategist_ab.py`, 3종목) 결과 품질 동등 이상 + 비용 76%↓. max_tokens 1500→2560(truncation 방지). 딥다이브 389→~175원, 손익분기 월48회.
- **prompt caching 진단**: 전 에이전트 system 1024자+로 캐싱 이미 적용. cache_read=0은 트래픽 패턴(5분 TTL) — 코드 정상, 트래픽 후 재평가.
- **응답캐시 L2 영속화**(`utils/cache.py`): L1메모리+L2 Firestore(`ai_response_cache`, TTL 6h). **캐시 키가 messages(현재가·MA·"기준 시각") 해시라 데이터 갱신 시 자동 miss → 신선도 구조적 보장**.
