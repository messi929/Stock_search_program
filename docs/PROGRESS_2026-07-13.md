- 정가 39,000/398,000은 2026-06-29 인상분(`b3a87d3`)으로, **실제로 청구되던 가격**이다.

### 문구 정정 (중요)
기존 "연 구독은 **약 2개월치 할인**" 문구는 월가가 29,000이 되면 **사실이 아니다**
(29,000 × 12 = 348,000 vs 319,000 → 차액 29,000원 = 정확히 **1개월치**).
→ 요금제 FAQ·약관 제4조를 "약 1개월치 절약"으로 정정. 할인율(20%/26%)은 정가 대비라 그대로 유효.

## 3. 검증 (배포 전·후)

LS API로 variant 실가격 직접 조회 — prod가 참조하는 ID와 동일함을 확인:

```
variant 1737014 (month) → price 2,900,000 (= 29,000원), trial 7일
variant 1737002 (year)  → price 31,900,000 (= 319,000원), trial 7일
```
(LS는 최소 화폐단위×100으로 저장 → /100 해석. 100배 오류 아님.)

- 백엔드 `/api/lemon-config` → `{"enabled":true,"variantMonthly":"1737014","variantYearly":"1737002"}` — 표시가와 청구가 일치.
- 라이브 `axislytics.com/pricing` — 29,000 / 319,000이 본 가격, 39,000 / 398,000은 취소선, 26%·20% 배지 렌더링 확인.
- `tsc --noEmit` 통과.
