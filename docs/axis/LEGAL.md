## 🚨 절대 원칙 (Hard Rules)

Axis는 **유사투자자문업 신고 없이** 운영하므로, 다음을 반드시 준수:

### 1. "추천" 단어 절대 금지

**모든 코드, 프롬프트, UI에서 다음 단어 사용 금지:**

❌ **금지 단어 리스트**
```
추천합니다 / 추천드립니다
사세요 / 매수하세요 / 매도하세요
매수 신호 / 매도 신호 / 진입 신호
유망합니다 / 유망주
목표가 / 매수가 / 적정가
사야 합니다 / 팔아야 합니다
"이 종목은 좋다/나쁘다" 식 단정
"확실히 오릅니다" / "반드시 오릅니다"
```

✅ **대체 단어 리스트**
```
"분석 결과..."
"관찰 구간"
"참고 수치"
"이 데이터로 판단해보세요"
"검토 시그널"
"관찰 가치가 있습니다"
"참고 범위"
"본 정보를 참고하시기 바랍니다"
```

### 2. 모든 AI 응답 면책 문구 필수

**모든 AI 분석/응답 끝에 다음 문구 자동 삽입:**

```
📌 본 분석은 투자 권유가 아닌 정보 제공입니다.
   최종 투자 판단은 사용자 본인의 책임입니다.
   Axis는 자본시장법상 투자자문업 면허가 없으며,
   특정 종목의 매매를 권유하지 않습니다.
   투자에는 원금 손실 위험이 있습니다.
```

### 3. UI에 명시적 표시

**모든 분석 결과 페이지 상단/하단:**

```
⚠️ 이 서비스는 투자 분석 도구입니다.
   투자 자문이나 종목 추천 서비스가 아닙니다.
```

### 4. 페르소나 프롬프트에도 내장

각 페르소나 시스템 프롬프트에 명시:
```
"당신은 분석가이지 추천자가 아닙니다.
'추천', '사세요', '매수' 단어를 절대 사용하지 마세요.
모든 결론은 사용자가 판단할 수 있도록 정보 제공만 하세요."
```

## ⚖️ 한국 법률 배경

### 자본시장법 관련 조항

**제17조 (무인가 영업행위 금지)**
- 금융투자업(투자자문업 포함)을 인가/등록 없이 영위 금지

**유사투자자문업** (자본시장법 제101조)
- 불특정 다수에게 투자 조언 → 신고 대상
- 자본금 1억원 이상 (개인사업자도 가능)
- 신고 절차: 금융위원회 → 약 30일

### Axis가 신고 안 해도 되는 이유

1. **"투자 조언"이 아닌 "정보 제공"**
   - 특정 종목 매수/매도 권유 X
   - 객관적 데이터 정리 + 사용자 판단 강조

2. **개별 맞춤 자문 X**
   - 모든 사용자에게 같은 분석 도구 제공
   - 1:1 상담 X

3. **수익률 보장 X**
   - 어떠한 수익률도 약속 안 함
   - 모든 결과는 "참고" 수준

### ⚠️ 회색지대 — 주의 필요

**위험 신호 (이런 표현이 있으면 신고 대상 가능성)**

❌ **위험한 패턴**
```
"이번 주 추천 종목"
"수익률 +20% 기대"
"회원님께 적합한 포트폴리오"
"전문가 추천 리스트"
```

✅ **안전한 패턴**
```
"이번 주 관찰 가치 종목"
"이 데이터로 판단 시..."
"모든 사용자에게 같은 분석 도구"
"객관적 데이터 정리"
```

## 🛡 코드 레벨 안전장치

### 1. 프롬프트 자동 검증

```python
# utils/prompt_validator.py

FORBIDDEN_WORDS = [
    "추천합니다", "추천드립니다", "추천드려요",
    "사세요", "매수하세요", "매도하세요",
    "매수 신호", "매도 신호", "진입 신호",
    "유망합니다", "유망주",
    "목표가", "매수가", "적정가",
    "사야 합니다", "팔아야 합니다",
    "확실히", "반드시", "분명히 오릅니다"
]

def validate_response(text: str) -> tuple[bool, list[str]]:
    """AI 응답에서 금지 단어 검출"""
    found = [w for w in FORBIDDEN_WORDS if w in text]
    is_clean = len(found) == 0
    return is_clean, found

# 사용 예시
def post_process_ai_response(response: str) -> str:
    is_clean, forbidden_found = validate_response(response)
    
    if not is_clean:
        # 로그 + 알림
        logger.error(f"금지 단어 발견: {forbidden_found}")
        # 자동 수정 시도 (또는 에러)
        for word in forbidden_found:
            response = response.replace(word, "[필터링됨]")
    
    # 면책 문구 자동 추가
    response += "\n\n" + DISCLAIMER
    return response
```

## 📜 이용약관 핵심 조항

### Article X. 서비스 성격

```
1. Axis(이하 "본 서비스")는 투자 정보 제공 및 분석 도구입니다.
2. 본 서비스는 자본시장법상 투자자문업 또는 투자중개업 면허를 보유하지 않습니다.
3. 본 서비스에서 제공하는 모든 분석, 데이터, AI 응답은 투자 권유가 아닌 정보 제공입니다.
4. 사용자는 본 서비스 정보를 참고용으로만 사용하며, 모든 투자 결정은 사용자 본인의 책임입니다.
```

### Article Y. 면책

```
1. 회사는 본 서비스에서 제공하는 정보의 정확성, 완전성, 적시성을 보증하지 않습니다.
2. 사용자가 본 서비스 정보를 기반으로 한 투자로 인해 발생한 손실에 대해 회사는 책임지지 않습니다.
3. 회사는 AI 모델의 한계로 인해 잘못된 정보가 제공될 수 있음을 사용자에게 고지합니다.
4. 사용자는 투자 결정 시 반드시 다른 정보원을 함께 참고하고, 필요시 면허 보유 전문가와 상담해야 합니다.
```

### Article Z. AI 응답의 한계

```
1. 본 서비스는 Anthropic의 Claude AI 모델을 활용합니다.
2. AI 응답은 학습 데이터의 한계, 추론 오류, 데이터 신선도 문제 등으로 부정확할 수 있습니다.
3. 사용자는 AI 응답을 절대적으로 신뢰하지 말고, 본 서비스의 "검증" 기능을 활용하여 데이터를 재확인할 것을 권고합니다.
4. AI는 시장 미래를 예측할 수 없으며, 본 서비스의 모든 분석은 과거 데이터 기반입니다.
```

```python
"""배포 전 법적 안전 자동 검사"""

import os

from pathlib import Path

FORBIDDEN_WORDS = [
    "추천합니다", "추천드립니다",
    "사세요", "매수하세요", "매도하세요",
    "매수 신호", "매도 신호",
    "유망합니다", "유망주",
    "목표가", "매수가",
]

def check_files():
    """모든 코드/문서에서 금지 단어 검사"""
    issues = []
    
    # 검사 대상
    targets = [
        Path("agents"),
        Path("personas"),
        Path("api"),
        Path("web/app"),
        Path("web/components"),
    ]
    
    for target in targets:
        if not target.exists():
            continue
        for file in target.rglob("*"):
            if file.suffix in [".py", ".ts", ".tsx", ".md"]:
                content = file.read_text()
                for word in FORBIDDEN_WORDS:
                    if word in content:
                        # 주석에서 언급한 건 OK
                        lines = content.split("\n")
                        for i, line in enumerate(lines, 1):
                            if word in line and not line.strip().startswith("#") and not line.strip().startswith("//"):
                                issues.append(f"{file}:{i}: {word}")
    
    if issues:
        print("🚨 금지 단어 발견:")
        for issue in issues:
            print(f"  {issue}")
        return False
    
    print("✅ 모든 파일 통과")
    return True

def check_disclaimer():
    """면책 문구 누락 검사"""

    pass

if __name__ == "__main__":
    if not check_files():
        exit(1)
```
