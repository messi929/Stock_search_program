## 시작하기

### 1) 로그인
- Axis URL 접속 → "로그인" → Google 계정 선택
- 첫 진입 시 4문항 온보딩 (1분 소요): 투자 경력 / 관심 섹터 / 투자 원칙 / 선호 페르소나

### 2) 종목 분석 (가장 중요)
1. 상단 검색 또는 `/screener`에서 카테고리 클릭 → 종목 선택
2. 종목 페이지에서 4개 에이전트 결과가 차례로 채워짐 (약 60-90초)
   - **Research**: 시황·뉴스·매크로
   - **Analyst**: 기술적·펀더멘털 해석
   - **Validator**: ⭐ **실시간 검증** — 가격·PER·PBR·ROE를 현재 데이터로 재조회. 신뢰도 점수와 Contrarian 시나리오 제시.
   - **Strategist**: 페르소나 관점 종합 + 관찰 구간

### 3) 페르소나 전환
- 분석 페이지 상단 탭으로 블랙록 / ARK / 그레이엄 전환
- 같은 종목을 다른 페르소나가 어떻게 평가하는지 비교
- (Free: 블랙록만, Pro: 3종 모두)

### 4) 검증 버튼
- Validator 결과의 "다시 검증" 버튼 → 가격이 stale 했을 때 재조회
- 첫 호출 후 1시간 캐싱 → 같은 입력은 0원

### 5) 관심 종목 + 진입선
- 종목 분석 후 "⭐ 관심 추가"
- 1차/2차/3차 관찰 구간 저장
- 알림은 v1.1 (베타 기간 중에는 토글만 미리 켜두실 수 있음)

### 6) 커스텀 스크리너 (Pro)
- `/screener/custom` — PER·PBR·ROE·RSI 등 직접 조합
- 최대 20개까지 저장

## FAQ

**Q. 종목을 추천해주나요?**
> 아니요. Axis는 정보 제공 도구이며 종목 추천이나 매매 신호를 제공하지
> 않습니다. 모든 결정은 사용자 본인의 책임입니다.

**Q. AI가 알려준 수치를 그대로 믿어도 되나요?**
> 안 됩니다. AI는 환각이 발생할 수 있습니다. 분석 화면 상단의 "검증"
> 버튼으로 모든 수치를 실시간 데이터와 비교 후 사용하세요.

**Q. 분석이 너무 느립니다.**
> 4개 에이전트가 차례로 작동하므로 첫 호출은 60-90초가 정상입니다.
> 같은 종목 재호출은 캐시 히트로 0.1초입니다. 페르소나만 바꿀 때는
> Strategist만 재실행됩니다 (~10초).

**Q. 데이터가 오늘 거래일과 안 맞아 보입니다.**
> 한국 종목은 장 마감 후 1-2시간 안에 갱신됩니다. 휴장일 또는 장 시작
> 직후에는 직전 거래일 데이터가 표시됩니다.

**Q. 베타 종료는 언제인가요?**
> 미정입니다. 충분한 피드백이 모이고 핵심 이슈가 해결되면 정식 런칭을
> 공지합니다. 베타 사용자에게는 미리 알려드립니다.
